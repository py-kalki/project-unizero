# UNIZERO

## What This Is

UNIZERO is a comprehensive AI discovery and subscription management platform. A one-stop destination to explore ALL AI tools in the market (from startups to big tech), compare them by pricing, features, and use cases, manage personal and team AI subscriptions, track spending, and in the future — resell AI subscriptions directly like a marketplace.

## Core Value

**One platform to discover, compare, and manage all your AI tools — and eventually buy/sell them in one place.**

## Requirements

### Active

- [ ] Explore AI tools by category (LLMs, Image/Video, Productivity, Audio, Coding, etc.)
- [ ] View comprehensive AI tool listings with detailed information
- [ ] Compare AI tools side-by-side (pricing, features, use cases)
- [ ] User authentication (sign up, sign in, session management)
- [ ] Add personal AI subscriptions to track
- [ ] View and manage added subscriptions
- [ ] Set payment reminders for subscriptions
- [ ] Dashboard showing total monthly AI spending
- [ ] Spending breakdown by category
- [ ] Interactive comparison UI for AI tools
- [ ] Search and filter AI tools

### Out of Scope

- [ ] Business/team features — deferred to v2
- [ ] AI reselling/marketplace — deferred to v2+
- [ ] AI recommendation engine — deferred to v2
- [ ] User reviews/ratings — deferred to v2
- [ ] API integrations for automatic subscription tracking — deferred to v2

## Context

- **Greenfield project** — building from scratch
- **No existing data** — will need to populate AI tools database
- **Dual audience** — individuals first, businesses later
- **Inspiration:** Steam, Epic Games Store, Amazon — marketplace model for AI tools

## Constraints

- **Timeline:** No fixed deadline — quality over speed
- **Data:** Need comprehensive AI tools database — will require research and manual curation
- **Tech:** Modern, scalable stack — open to recommendations
